EasyMock Class Extension

This extension allows generating Mock Objects for classes. It has been contributed by Joel Shellman, Henri Tremblay and Chad Woolley. Henri Tremblay maintains this extension.

  • EasyMock 2.4 Class Extension (2008-08-30)
  • EasyMock 2.3 Class Extension (2007-12-26)
  • EasyMock Class Extension 1.2.1 (2007-03-21)
  • For other downloads, please go to the EasyMock CE downloads section at SourceForge.
